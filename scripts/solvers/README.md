# Scripts used to install solvers


# Files
```
├── config_aiger.sh         :   config for aiger
├── config_bitwuzla.sh      :   config for bitwuzla
├── config_boolector.sh     :   config for boolector
├── config_kissat.sh        :   config for kissat
├── config_solvers.sh       :   config for all solvers
└── install_solvers.sh      :   main script install solvers
```

# Usage
```
./install_solvers.sh --[solver_name]
./install_solvers.sh --aiger
```
