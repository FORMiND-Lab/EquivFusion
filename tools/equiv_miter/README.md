# equiv_miter
## a Miter tool

### Usage
```equiv_miter [options] <input file> [input file]```

### Options
- `--c1=<module name>` specifies a module name for the first circuit
- `--c2=<module name>` specifies a module name for the second circuit
- output type
  - `--smtlib`  miter to smtlib output
  - `--aiger`   miter to aiger output
  - `--btor2`   miter to btor2 output
